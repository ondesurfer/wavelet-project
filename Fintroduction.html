This site provides you with a database of scaling functions for wavelets.

You can filter certain ones out by specifiying properties like symmetry, polynomial exactness, critical Hoelder and Sobolev exponents or spline order (in the case of spline functions).

Thereby the critical Hoelder exponent is defined as


$\tilde{\beta}(\phi) := \sup \{\, \gamma \geq 0 \, ; \, \gamma = k + \alpha \, ; \, k \in \mathbb{N} \, , \, \alpha \in (0, 1] \, ; \, \phi \in \mathcal{C}^{k, \alpha}(\mathbb{R}) \,\}.$

and the critical Sobolev exponent is defined as


$\beta(\phi) := \sup \{\, \alpha \geq 0 \, ; \, \phi \in H^{\alpha}_2(\mathbb{R}) \, \}.$
